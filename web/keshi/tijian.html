万安体检中心简介

万安医院体检中心成立于2017年5月，依托医院强大的专家阵容和各种先进的医疗设备而建立，拥有一支高素质的体检队伍，由具有丰富临床保健经验和体检经验的20名医护人员组成；同时配有专门的导检人员，提供热情、周到的服务，在最短的时间内获得最满意的体检效果。

本中心拥有一流的医疗设备，MX16-SLICE飞利浦全身螺旋CT进口、 数字化PLD5600X线机、Apogee全数字多普勒超声彩色Ｂ超机、 超声经颅多普勒血流分析仪、动态心电图、EXC-90数码电子阴道镜、LEEP刀、全自动生化分析仪、自动血凝血纤仪、全自动微生物鉴定系统、血气分析仪、15项自动血球计数仪、心功能测定仪、放射免疫γ－计数仪等。


为了保障您健康体检的顺利进行和检查结果的准确有效，请您认真阅读以下内容：



一、健康体检前：
1. 体检前3天内不要吃含脂过高或油炸性食物，禁食含血制品及含铁类食物，应清淡饮食，勿饮酒、勿劳累。体检前日晚保持
    7-8小时睡眠，晚12点后禁食。
2. 行幽门螺杆菌检查者12小时内请勿饮水，并确保一个月内未服用抗生素及铋剂（如：丽珠得乐、胃得乐、胃必治等）。
3. 受检者放射科检查：
（1）当您行胸片检查时，请您摘掉并妥善保管项链及胸针等物品；勿穿金属钮扣上衣；女性受检者勿穿戴有金属钢圈及金属
        饰物的内衣，以免影响您的检查结果。
（2）行核磁检查时，应取下一切金属物品，若体内有金属（如：金属假牙、心脏起搏器、节育环、克氏钉、钢板等），
        请告知医务人员。
（3）已受孕者及备孕者的受检者，请您不要做放射科的检查及放射骨密度检查。
4. 腹部B超检查需空腹，盆腔B超或前列腺B超，请您需憋尿后做检查。
5. 便标本采集，最好在体检当日留取大便标本，如果您不方便在体检中心留取大便标本，可先在家留好，用清洁密闭容器装
    好后2小时内送到体检中心即可。
6. 特别受检者人群：
（1）高血压、心脏病、哮喘等慢性病患者检查当日请您正常服药，清晨可少量饮水服用（小于100毫升，幽门螺杆菌检查者
        除外），其它慢性疾病患者请将平时服用的药物携带备用。
（2）戴隐形眼镜者请改为框架眼镜。
（3）如您患有传染性疾病或其他急性疾病，为保障您和他人的健康，请务必痊愈后再择期参检。
7. 女性受检者：
（1） 怀孕或可能已受孕的女士，勿做肛诊及妇科检查。
（2） 体检时请避开月经期，建议在月经结束后三天进行体检，体检当日所穿衣物方便穿脱。
（3）体检前日请停止阴道用药，清水冲洗，不用洗液并避免性生活，以免影响 宫颈细胞检查的准确性。
（4）妇科医生检查前请您告知查体医生是否有妇科手术史；医生检查过程中和检查后可能出现少量出血。
（5）无性生活史者禁做妇科检查；未婚要求做妇科检查者，请本人签字；未满18岁女士要求妇科检查，需监护人签署同意书。
二、受检者体检当日：
1. 请您于上午7:30-10：00到体检中心按预约时间前来健康体检。
2. 健康体检当日请您携带身份证、护照、军官证等有效证件。
3. 如您个人信息发生变化，体检当日出示有效证件，便于更改信息。
4. 若您需要变更体检日期或退检，务必提前以电话或亲临万安体检中心办理相关手续。
三、受检者检查后：
1. 为保障良好的体检秩序，请您妥善安排好体检当日的其他事宜，若体检过程中因本人放弃的项目，请您在导引单签字，
    望您理解。
2. 团检单位的报告由单位统一领取。
3. 妇科检查者3~5个工作日后，来本体检中心领取，个体受检者报告,报告领取时间：
        下午2:30-5:30。
4. 您也可以在网上便捷查询您的体检报告，可以 点击此处 了解"如何在手机上直接查看我的体检报告？"。
5. 体检报告中如有异常检查结果，我们会以电话或当面等方式与您取得联系。

体检中心配备功能齐全的体检软／硬件，专家运用软／硬件对体检结果进行分析和把关，给体检者提供治疗意见。本中心免费为体检者建立个人健康档案，随时为体检者提供健康资料，恪守为体检者保密承诺。为满足不同体检人群的需求，设计了多种体检套餐可供选择。
